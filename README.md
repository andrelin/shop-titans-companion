# Shop Titans Companion

A small web app of personal tools for Shop Titans, built on the community-maintained game-data spreadsheet.

The first tool is a **Dragon Invasion item-power ranker** that lists every craftable item by airship power, with and without enchantments, applying affinity bonuses and quality multipliers.

Live site: https://andrelin.github.io/shop-titans-companion/

## Development

```sh
npm install
npm run sync   # download latest CSVs from the canonical data spreadsheet
npm run dev    # start Vite dev server
npm run build  # type-check + production build to dist/
```

## How it works

- **Data source.** The canonical Shop Titans data spreadsheet (`1WLa7X8h3O0-aGKxeAlCL7bnN8-FhGd3t7pz2RCzSg8c`) is downloaded as CSV at build time by `scripts/sync-data.ts`, parsed into typed JSON, and committed to `data/`. A GitHub Action re-runs the sync on a schedule and opens a PR / commits if the data has changed.
- **Power formula.** Verified against the community Dragon Invasion sheet's published rankings:
  - `basePower = 0.8 · atk · (1 + 10·crit) + 1.2 · def · (1 + 10·eva) + 5 · hp`
  - Quality multipliers scale each underlying stat (Common 1, Superior 1.25, Flawless 1.5, Epic 2, Legendary 3).
  - Enchantments add a flat per-stat amount to airship points; affinity-matched enchants give the higher of two values from the per-tier table.
  - The "+25% Bonus Airship Power" Starforged Milestone is detected per blueprint and applied as a multiplier when toggled on.
- **Roadmap.** See [`docs/future-tools.md`](docs/future-tools.md) for the list of additional tools we might add (quest planner, hero loadouts, crafting-upgrade ROI, etc.).

## Layout

```
  tools/

.github/workflows/    pages deploy + scheduled data sync
```
